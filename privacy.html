TriggerTemplate Privacy Policy

Last updated: May 10, 2026

Data Collected

TriggerTemplate does not collect, sell, transmit, or share personal information with the developer or third parties.

Data Stored

Templates that you create are stored using Chrome extension storage:

  • chrome.storage.sync stores your template names, trigger actions, template text, and template order.
  • chrome.storage.local stores local panel preferences such as side-tab position, open/closed state, and whether the side tab is hidden.

If Chrome Sync is enabled in your browser profile, Chrome may sync template data across browsers signed into the same Chrome account. TriggerTemplate does not operate its own server.

Website Access

The extension injects a content script into webpages so it can detect trigger text in editable fields and replace it with your saved template. Template expansion happens locally in your browser.

Remote Code

TriggerTemplate does not load or execute remote code.

Contact

For support, use the GitHub repository: TriggerTemplate on GitHub.
